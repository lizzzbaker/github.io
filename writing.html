Addressing Healthcare Inequality through Predictive Analytics

This project investigates how geography, income, and hospital quality shape unequal health outcomes across the United States. Using large federal datasets, we engineered a machine-learning model that predicts a community’s healthcare “risk level” by ZIP code. Capturing factors like patient satisfaction, infection rates, and local provider availability, Our model achieved 91% accuracy. It also powers an interactive U.S. map, allowing users to visualize healthcare disparities in real time. The work blends data science and public policy to offer a practical tool for policymakers and advocates seeking more equitable healthcare systems.

Open-Source Contribution: Improving MuseScore’s Governance & Developer Experience

In this project, we contributed to MuseScore, one of the world’s most widely used open-source music notation platforms. We focused on two improvements: enhancing the UI by replacing a confusing “[LINK]” label with a clean icon and dynamic tooltip, and creating new developer documentation that fixes a common build error in Visual Studio. These contributions required navigating a large C++/Qt codebase, performing UI updates in QML, running visual regression tests, and writing onboarding-ready technical guides. The work strengthened the project’s accessibility for both users and new contributors, and the maintainers invited parts of it into the official MuseScore Wiki.
